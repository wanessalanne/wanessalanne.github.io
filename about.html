Sobre nós

Nossa História

Eru mobly foi um projeto desenvolvido visando implementar os conhecimentos adquiridos em HTML, CSS, JAVASCRIPT, JQUERY e BOOTSTRAP com a finalidade de criar um site com o tema "móveis" tendo como requisito o uso de Jquery para a função de um botão(seta) no canto direito da página para retornar suavemente ao topo, o uso de Javascript para as validações de um formulário, estilização do rodapé utilizando somente CSS, utilizar a tag aside e article no Html, realização de comentários com o indice dos requisitos na linha 4, criação de formulário, menu dropdown, estilo dos botões, texto e layout responsivo com Bootstrap.
